# Calculator DevOps Pipeline

## 📌 Problem Statement
This project aims to build a simple calculator application and implement a complete DevOps CI/CD pipeline for automated build, testing, and deployment.

## 🎯 Objectives
- Develop a Flask-based calculator application
- Implement version control using Git
- Automate build and testing using CI/CD
- Containerize the application using Docker
- Deploy the application to a cloud platform

## 🚀 Features (Planned)
- Basic arithmetic operations (Add, Subtract, Multiply, Divide)
- Error handling (division by zero)
- Web-based UI

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- GitHub Actions (CI/CD)
- Docker
- Render (Deployment)
